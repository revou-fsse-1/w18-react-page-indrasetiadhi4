# Photo Club Gallery

## Installation

```bash
# Clone the repository
$ git clone https://github.com/revou-fsse-1/w18-react-page-indrasetiadhi4.git

# Change directory
$ cd w18-react-page-indrasetiadhi4

# Install Dependencies
$ pnpm install

# Run app
$ pnpm run dev
```
